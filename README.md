# CLBOT RAG

## Overview

CLBOT RAG is a project designed to build a powerful chatbot leveraging retrieval-augmented generation techniques. The primary goal of this project is to enhance the capabilities of chatbots by providing them with up-to-date and relevant information during interactions.

## Features

- **Dynamic Responses**: Generate responses based on the latest information from various data sources.
- **Multi-Source Integration**: Combine data from multiple platforms to give users comprehensive answers.
- **User-Friendly Interface**: A simple and intuitive interface for seamless interaction.

## Installation

To get started with CLBOT RAG, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/saadbinather/CLBOT_RAG.git
   cd CLBOT_RAG
   ```
2. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the chatbot, execute the following command after installation:
```bash
python main.py
```

After starting the bot, you can interact with it via the terminal or connect it to your preferred messaging platform.
